Reciplease

Reciplease est une application iOS qui vous permet

  • d'ajouter les ingrédients que vous avez dans une liste
  • parcourir les recettes résultantes de cette liste
  • d'enregistrer l'une d’elle dans les favoris

Technique

  • Persistance des données (CoreData)
  • Appel réseau avec l’API Edamam
  • Gestion de listes, animations (TableView, UIKit)
  • Cocoapods / Alamofire
  • Tests unitaires, architecture MVC

Application non disponible sur l'AppStore
